# HyADR
Drug adverse effect prediction using a novel heterogenous hypergraph embedding
# Usuage
python main.py

# Input
Node features for each type, known edges, randomly generated unknown hyperedges, sparse normalized adjacency matrix.
# Output
Entities embedding
